# Lab_2_terminal

# Мини-оболочка с файловыми командами

## Описание

Мини-оболочка, реализующая основные команды управления файлами и каталогами средствами Python: просмотр, копирование, перемещение, удаление, работа с архивами (zip/tar), история команд, undo и поиск по содержимому

## Установка:

```bash
git clone https://github.com/SOKLY6/Lab_2_terminal.git
cd Lab_2_terminal
```

## Запуск:

```bash
python3 src/main.py
```

# Функционал:

## Файловые операции

### `cat [файлы...]`
- **Вход**: список путей к файлам
- **Действие**: выводит содержимое файлов в терминал
- **Пример**: `cat file1.txt file2.txt`

### `ls [флаги] [пути...]`
- **Вход**: флаги `-l` (подробно), `-a` (скрытые) и пути
- **Действие**: показывает список файлов/папок
- **Пример**: `ls -la ~/Documents`

### `cp [-r] [пути для копирования...] [путь-назначение]`
- **Вход**: флаг `-r` (рекурсивно) и пути
- **Действие**: копирует файлы/папки
- **Пример**: `cp -r folder/ backup/`

### `mv [пути для перемещения...] [путь-назначение]`
- **Вход**: исходные пути и путь назначения
- **Действие**: перемещает или переименовывает
- **Пример**: `mv old.txt new.txt`

### `rm [-r] [пути для удаления...]`
- **Вход**: флаг `-r` (рекурсивно) и пути
- **Действие**: удаляет в корзину
- **Пример**: `rm -r temp/`

## Навигация

### `cd [директория]`
- **Вход**: путь к директории
- **Действие**: меняет текущую директорию
- **Пример**: `cd ~/Documents`

### `history [N]`
- **Вход**: число N
- **Действие**: показывает последние N команд
- **Пример**: `history 10`

## Поиск

### `grep [-r -i] [шаблон] [файлы...]`
- **Вход**: флаги `-r` (рекурсивно), `-i` (игнорировать регистр), шаблон (регулярные выражения) и пути
- **Действие**: ищет текст в файлах
- **Пример**: `grep -ri "error" logs/`

## Архивация

### `tar [архив] [файлы/папки...]`
- **Вход**: имя архива и файлы для упаковки
- **Действие**: создает tar.gz архив
- **Пример**: `tar project.tar.gz src/ docs/`

### `untar [архивы...] [директория]`
- **Вход**: архивы и директория для распаковки
- **Действие**: распаковывает tar.gz
- **Пример**: `untar backup.tar.gz`

### `zip [архив] [файлы/папки...]`
- **Вход**: имя архива и файлы для упаковки
- **Действие**: создает zip архив
- **Пример**: `zip photos.zip *.jpg`

### `unzip [архивы...] [директория]`
- **Вход**: архивы и директория для распаковки
- **Действие**: распаковывает zip
- **Пример**: `unzip data.zip ~/downloads/`

## Система

### `undo`
- **Вход**: нет аргументов
- **Действие**: отменяет последнюю операцию cp/mv/rm
- **Пример**: `undo`
